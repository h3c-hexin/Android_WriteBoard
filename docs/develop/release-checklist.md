# 白板 发布前检查清单

> 每次发版前在真机上过一遍，通过后再 commit release。

---

## 核心画布

- [ ] 画笔书写流畅，无卡顿
- [ ] 切换笔刷粗细、颜色后书写正常
- [ ] 橡皮擦分段正确（擦中间，两端保留）
- [ ] 撤销 / 重做还原正确

## 图形工具

- [ ] 6 种图形（直线、箭头、矩形、圆形、三角形、菱形）绘制正常
- [ ] 图形撤销正常

## 多页管理

- [ ] 新增、删除、切换页面正常
- [ ] 不同页面背景（空白 / 网格 / 横线）显示正确
- [ ] 页面缩略图渲染正常

## 文件 & 数据

- [ ] 保存白板 → 重新打开，内容一致
- [ ] 强杀 App 重启，自动恢复上次内容
- [ ] 新建白板后强杀重启，不恢复旧内容

## 扫码分享

- [ ] 生成二维码正常，手机扫码可访问
- [ ] 全屏放大二维码可用

## LAN 协同

- [ ] 主设备开启协同，子设备扫码加入成功
- [ ] 子设备加入瞬间，页码跟随主设备当前页
- [ ] 多端同步书写正常
- [ ] 多端同步擦除正常
- [ ] 子设备切后台断连，重新加入，设备列表无旧记录
- [ ] 主设备结束协同，子设备收到提示

## 发版操作

```
1. 更新 app/build.gradle.kts：versionCode +1，versionName 改为新版本号
2. ./gradlew assembleRelease
3. 安装到测试设备，过以上清单
4. git commit "release: vX.Y.Z — 变更简述"
5. git push
```
