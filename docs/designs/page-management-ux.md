## 1. 页面管理子面板总体规范

### 1.1 触发入口

- **工具栏页码区**：点击 `◀ 2/5 ▶` 中间的页码文字区域（触控热区最小 `80dp × 64dp`）
- 翻页按钮 `◀` / `▶` 不触发子面板，仅执行翻页
- 再次点击页码区域 → 关闭子面板（切换显示）

### 1.2 Popup 规范

页面管理子面板为 **Compose `Popup`**，遵循 `toolbar-component.md §0` 二级子面板统一规范：

```
┌──────────────────────────────────────────────────────────┐
│  页面管理                                    3 页          │
│                                                          │
│  ┌────────┐  ┌────────┐  ┌────────┐  ┌────────┐  ←可左右滑
│  │[◀][▶] │  │[◀][▶] │  │[◀][▶] │  │        │
│  │  缩略图  │  │  缩略图  │  │  缩略图  │  │   ＋    │
│  │        │  │        │  │        │  │  新增页  │
│  │   [🗑] │  │   [🗑] │  │   [🗑] │  │        │
│  └────────┘  └────────┘  └────────┘  └────────┘
│     [1]          [2]          [3]
│  [空白][网格][横线] ...（背景切换标签）
└──────────────────────────────────────────────────────────┘
        ▲（Popup 底边 = 工具栏顶边，以页码区为锚点）
```

### 1.3 面板尺寸

| 属性 | 值 |
|------|---|
| 背景色 | `color.bottomsheet.background`（`#2D2D2D`） |
| 圆角 | `16dp`，**四角均匀**（无贴底半圆角） |
| 最大宽度 | `648dp`（约 3.5 个卡片宽，可左右滚动提示） |
| 高度 | 由内容决定（`wrapContentHeight`），无固定高度、无全屏遮罩 |
| 水平内边距 | `24dp` |
| 垂直内边距 | `16dp` |
| 阴影 | `shadowElevation = 16dp` |
| 遮罩 | **无**（画布保持可见） |

> **最大宽度 648dp 计算**：3.5 × (160dp 卡片 + 16dp 间距) - 16dp + 2 × 24dp = 648dp。
> 设计为 3.5 个卡片宽而非整 3 个，目的是让用户直觉感知列表可左右滚动。

### 1.4 关闭行为

| 关闭方式 | 行为 |
|---------|------|
| 点击面板外任意区域 | 关闭 Popup（`dismissOnClickOutside`） |
| 再次点击页码区域 | 关闭 Popup |
| 点击任意缩略图切换页面 | 切换页面后自动关闭 Popup |

### 1.5 打开时初始滚动位置

面板打开时，当前页缩略图自动滚动至列表**水平居中**位置，便于用户感知当前页在整个页面序列中的位置。

## 2. 缩略图网格布局

### 2.1 缩略图尺寸

| 模式 | 缩略图宽度 | 缩略图高度 | 比例 |
|------|---------|---------|------|
| 大屏 | `160dp` | `90dp` | 16:9（横屏画布比例） |
| PAD | `120dp` | `68dp` | 16:9 |

缩略图之间的间距：`spacing.lg`（16dp）

### 2.2 缩略图样式

```
缩略图容器：
  圆角：radius.sm（8dp）
  边框（普通页）：1dp，color.divider（#424242，深色模式）
  边框（当前页）：2dp，color.brand.primary（#90CAF9，深色模式）
  背景：缩略图位图（异步渲染，变更时自动更新）
  加载中占位：浅灰矩形 + 居中 Loading 圆圈

当前页高亮：
  蓝色外描边（2dp，color.brand.primary）
  底部居中显示当前页标识文字（见 7.1 节）
  页码标签背景：color.brand.primary，圆角 radius.sm
```

### 2.3 缩略图上方控件：排序箭头

每张缩略图卡片的**右上角**固定显示两个箭头按钮：

```
  ┌───────────────────────────────────┐
  │ [▲ 上移] [▼ 下移]                 │   ← 右上角排列
  │                                   │
  │         缩略图内容                  │
  │                                   │
  │                        [ 🗑 删除 ] │  ← 右下角
  └───────────────────────────────────┘
```

- 箭头按钮触控热区：`44dp × 44dp`（大屏）/ `36dp × 36dp`（PAD）
- 图标尺寸：`20dp`（大屏）/ `16dp`（PAD）
- 图标颜色：`color.icon.on-surface`（`#EEEEEE`，深色模式）
- 背景：半透明黑色 `rgba(0,0,0,0.45)`，圆角 `radius.xs`（4dp）
- 禁用状态：第一页的 ▲ 上移禁用；最后页的 ▼ 下移禁用（`color.icon.disabled`）

### 2.4 缩略图下方：删除按钮

每张缩略图卡片的**右下角**固定显示删除按钮：

- 图标：🗑 垃圾桶图标
- 触控热区：`44dp × 44dp`（大屏）
- 图标颜色：`color.semantic.error`（`#EF9A9A`，深色模式）
- 背景：半透明黑色 `rgba(0,0,0,0.45)`，圆角 `radius.xs`
- 最后一页时禁用（最少保留 1 页），图标颜色 `color.icon.disabled`

### 2.5 页码标签

每张缩略图下方居中显示页码：

```
  ┌─────────────────────┐
  │       缩略图         │
  └─────────────────────┘
           [ 1 ]     ← 页码标签

  普通页码：typography.size.caption，color.icon.default
  当前页码：typography.size.caption，color.brand.primary，加粗（semibold）
```

### 2.6 新增页卡片

列表最右侧固定放置一张"新增页"卡片：

```
  ┌─────────────────────┐
  │                     │
  │         ＋          │  ← 居中显示加号
  │       新增页         │
  │                     │
  └─────────────────────┘
```

- 背景：`color.toolbar.background`，边框虚线 `2dp color.divider`，圆角 `radius.sm`
- `＋` 图标：`40dp`（大屏），颜色 `color.brand.primary`
- 文字 "新增页"：`typography.size.label`，`color.brand.primary`
- 禁用态（已达 20 页上限）：图标颜色 `color.icon.disabled`，虚线边框变浅

## 3. 新增页

### 3.1 触发入口

| 入口 | 位置 | 行为 |
|------|------|------|
| 工具栏 ＋ 按钮 | 工具栏 F 区最右侧 | 在当前页末尾新增一页，立即切换至新页 |
| 页面管理 Popup 中"新增页"卡片 | 列表最右侧 | 在末尾新增一页，Popup 保持打开，滚动至新页 |

### 3.2 新增页流程

```
用户点击新增页（任意入口）
  │
  ▼
  检查当前页数是否已达上限（20 页）
  │
  ├─ 已达 20 页 → 按钮禁用，Toast 提示："已达最大页数（20 页）"，2s
  │
  └─ 未达上限 →
       创建新 PageData（空白笔画列表，继承当前页背景类型）
       追加到 PageDataList 末尾
       当前页索引切换到新页
       触发页面切换动画（向右翻页）
       底部工具栏页码更新
       若 Popup 打开：滚动至新缩略图，新缩略图短暂高亮（500ms 脉冲）
```

### 3.3 新增页的默认背景

新增页默认继承当前页的背景类型（纯白 / 网格 / 横线），不重置为默认。用户可在页面管理 Popup 中针对单页切换背景。

## 4. 删除页

### 4.1 触发入口

- 页面管理 Popup 中，每张缩略图右下角的 🗑 删除按钮

### 4.2 删除流程

```
用户点击缩略图右下角 🗑 删除按钮
  │
  ▼
  检查是否为最后一页
  │
  ├─ 是最后一页 → 按钮禁用，点击无响应（不弹提示，禁用视觉已充分表达）
  │
  └─ 非最后一页 →
       弹出删除确认对话框（AlertDialog，二次确认）：
       ┌───────────────────────────────────────────────────┐
       │  确认删除第 N 页？                                  │
       │  此操作无法撤销。                                   │
       │                                                   │
       │  [    取消    ]  [  删除  （color.semantic.error）] │
       └───────────────────────────────────────────────────┘

  用户点击【删除】：
    从 PageDataList 移除对应 PageData
    若被删除的是当前页：
      切换至相邻页（优先切换到前一页，若删除的是第一页则切换至新的第一页）
    页码更新，Popup 中缩略图列表更新
    删除动画：缩略图从列表中消失（缩小淡出，150ms）

  用户点击【取消】：
    关闭确认弹窗，返回页面管理 Popup
```

### 4.3 删除确认规范

- 形式：**内嵌覆盖层**，直接叠加在被点击的缩略图卡片上，不弹出居中弹窗（就近操作，视线不离开操作位置）
- 蒙层：`rgba(0,0,0,0.80)`，`matchParentSize`，覆盖整张卡片
- 确认文字：`"删除第 N 页？"`，`12sp`，白色
- 取消按钮：深灰背景 `#3A3A3A`，圆角 `4dp`，文字 `color.icon.default`
- 删除按钮：淡红背景 `rgba(239,154,154,0.2)`，圆角 `4dp`，文字 `color.semantic.error`（`#EF9A9A`）
- 确认期间该卡片的切换页面点击被禁用；其他卡片操作不受影响

## 5. 页面排序（▲▼ 箭头按钮）

### 5.1 设计原则

- 不使用拖拽手势（会议室多人大屏场景，手势易误触，且与书写手势冲突）
- 每张缩略图提供 ▲ 上移 / ▼ 下移 两个按钮，点击一次移动一个位置

### 5.2 排序操作流程

```
用户点击缩略图 [▲ 上移] 按钮（目标：将第 N 页移至第 N-1 页）
  │
  ▼
  检查是否为第一页（不可再上移）
  │
  ├─ 是第一页 → ▲ 按钮禁用，无响应
  │
  └─ 非第一页 →
       PageDataList 中 index N 与 index N-1 互换
       缩略图列表执行位置交换动画（200ms，水平位移）
       页码重新计算，工具栏页码更新
       （当前页随位置移动，保持内容不变）
       排序操作纳入 UndoStack，可撤销
```

### 5.3 排序动画

```
交换动画（两张缩略图同时执行）：
  第 N 页缩略图：translateX: 0 → -（缩略图宽度 + 间距），200ms
  第 N-1 页缩略图：translateX: 0 → +(缩略图宽度 + 间距），200ms
  缓动曲线：animation.easing.standard（cubic-bezier(0.4, 0.0, 0.2, 1)）
  动画期间两个按钮短暂禁用（防止连续快速点击导致视觉混乱）
```

### 5.4 边界状态

| 页面位置 | ▲ 上移状态 | ▼ 下移状态 |
|---------|---------|---------|
| 第 1 页 | 禁用（`color.icon.disabled`） | 可用 |
| 中间页 | 可用 | 可用 |
| 最后页 | 可用 | 禁用（`color.icon.disabled`） |
| 唯一一页 | 禁用 | 禁用 |

## 6. 页面切换动画

### 6.1 翻页动画规范

（参考 design-tokens.md 6.6 节）

| 属性 | 值 |
|------|---|
| 动画时长 | `200ms` |
| 缓动曲线 | `animation.easing.standard`（`cubic-bezier(0.4, 0.0, 0.2, 1)`） |
| 切换方式 | 水平淡入淡出 + 轻微位移（cross-fade + slight horizontal slide） |
| 向右翻（切换到下一页） | 旧页：`translateX: 0 → -40dp` + `opacity: 1 → 0`；新页：`translateX: +40dp → 0` + `opacity: 0 → 1` |
| 向左翻（切换到上一页） | 旧页：`translateX: 0 → +40dp` + `opacity: 1 → 0`；新页：`translateX: -40dp → 0` + `opacity: 0 → 1` |
| 从缩略图跳转 | 同向翻页规则（目标页 > 当前页 → 向右翻；目标页 < 当前页 → 向左翻） |

### 6.2 翻页期间的交互禁用

- 动画进行期间（200ms），工具栏翻页按钮 `◀▶` 短暂禁用（防止快速连续点击跳帧）
- 画布书写功能不受影响（动画不阻塞触摸事件处理）

### 6.3 协同场景的页面切换

- 协同会话中，Host 切换页面时，所有参与者的视图自动同步跟随
- 若 Host 开启了"主持人控制"模式，参与者无法自行翻页（工具栏翻页按钮和页码点击禁用）
- 参与者视图收到 Host 翻页指令时，同样执行翻页动画（方向与 Host 一致）

## 7. 页码显示规范

### 7.1 工具栏页码区格式

```
格式：[当前页码] / [总页数]
示例：2 / 5

字号：typography.size.caption（14sp 大屏 / 12sp PAD）
字重：typography.weight.semibold（600）
颜色：color.icon.default（普通态）/ color.brand.primary（按压态）
对齐：水平居中
```

### 7.2 页面管理 Popup 中的页码标签

每张缩略图下方的页码标签格式：

```
普通页：单数字（如 "1"、"2"、"3"）
当前页：数字 + 选中态高亮（见 2.5 节）

字号：typography.size.caption（14sp 大屏）
位置：缩略图正下方，垂直居中间距 spacing.xs（4dp）
```

### 7.3 总页数变化时的动画

- 新增页或删除页后，工具栏页码区的总页数数字执行 `淡入淡出` 更新（`opacity: 0 → 1`，`100ms`）
- 当前页码不执行动画，直接更新（即时响应）

## 8. 页面背景设置

### 8.1 背景类型

每页可独立设置三种背景之一：

| 背景类型 | Token | 视觉 |
|---------|-------|------|
| 纯白（默认） | `color.canvas.background.solid` | 纯色白板，深色模式下 `#1E1E1E` |
| 网格 | `color.canvas.background.grid.*` | 均匀方格，线条 `#424242`（深色模式） |
| 横线 | `color.canvas.background.rule.*` | 等间距横线，线条 `#616161`（深色模式） |

网格间距 / 横线间距：`32dp`（大屏）/ `24dp`（PAD）

### 8.2 背景切换入口

页面管理 Popup 中，每张缩略图卡片下方常驻显示背景类型切换标签行：

```
缩略图卡片下方：
  [空白] [网格] [横线]   ← 每张卡片各自独立的背景切换标签
```

- 三个标签横排（`44dp × 24dp`，圆角 `4dp`），点击即时生效，缩略图实时更新
- 当前背景类型显示激活态（`ActiveIconBackground` 底色 + `PrimaryBlueDark` 文字描边）

### 8.3 背景切换行为

- 背景切换**即时生效**，无动画（背景仅视觉参考，不影响笔画数据）
- 背景设置存储于 `PageData.backgroundType`，随 `.pb` 文件保存
- 切换背景操作**不纳入 UndoStack**（简化设计，背景变更为用户主动偏好设置）
