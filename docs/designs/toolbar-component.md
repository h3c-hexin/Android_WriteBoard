## 0. 二级子面板统一规范

> 本节为所有从工具栏按钮触发的二级子面板（橡皮擦清除面板、图形工具面板、颜色选择器、页面管理）提供**统一的弹出规则**。设计原则：子面板应紧贴触发按钮、不覆盖工具栏、不阻碍主要画布区域。

### 0.1 弹出形式

所有工具栏二级子面板均使用 **Compose `Popup`**（无全屏遮罩），不使用 `ModalBottomSheet`。

| 属性 | 规范 |
|------|------|
| 实现方式 | Jetpack Compose `Popup` + 自定义 `PopupPositionProvider` |
| 遮罩 | **无**（不覆盖画布，画布保持可见） |
| 焦点 | `focusable = true`，`dismissOnClickOutside = true` |
| 关闭方式 | 点击面板外任意区域自动关闭 |

### 0.2 弹出位置

```
定位规则（PopupPositionProvider.calculatePosition）：

  垂直方向（Y）：
    Popup 底边 = 工具栏顶边
    y = windowSize.height - toolbarHeightPx - popupContentSize.height
    （工具栏高度固定为 72dp）

  水平方向（X）：
    Popup 水平中心 = 触发按钮水平中心
    x = anchorBounds.center.x - popupContentSize.width / 2
    边界约束：coerceIn(8dp, windowWidth - popupWidth - 8dp)
    （防止 Popup 超出屏幕左右边缘）
```

示意图：

```
  ┌─────────────────────────────────┐
  │         二级子面板内容             │
  │                                 │
  └──────────┬──────────────────────┘
             │  Popup 底边 = 工具栏顶边
  ════════════╪═══════════════════════════ ← 工具栏顶边
  ▒▒ 工具栏 ▒▒│▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒
             ↑
        触发按钮（锚点）
```

### 0.3 外观样式

| 属性 | 规范 |
|------|------|
| 背景色 | `color.bottomsheet.background`（`#2D2D2D` / `BottomSheetBackground`） |
| 圆角 | `16dp`，**四角均匀**（无"贴底"半圆角） |
| 阴影 | `shadowElevation = 12dp`（小面板）/ `16dp`（大面板） |
| 内边距 | 水平 `24dp`，垂直 `16dp`（大面板）；水平 `20dp`，垂直 `16dp`（小面板） |

### 0.4 宽度规则

| 面板 | 最大宽度 | 实际宽度策略 |
|------|---------|-----------|
| 橡皮擦清除面板 | `280dp`（固定） | 内容宽度决定 |
| 图形工具面板 | `284dp`（固定） | `3列 × 72dp + 间距 + 内边距` |
| 颜色选择器 | `560dp` | `widthIn(max=560dp)` + `wrapContentWidth()` |
| 页面管理 | `648dp`（约 3.5 个卡片宽） | `widthIn(max=648dp)` + `wrapContentWidth()` |

> **页面管理最大宽度计算**：3.5 × (160dp 卡片 + 16dp 间距) - 16dp（末尾间距）+ 2 × 24dp（水平内边距）= 648dp。
> 设计为 3.5 宽而非整 3 宽，是为了让用户直觉感知列表可左右滑动。

### 0.5 锚点放置规则

Popup 必须放置在**触发按钮的 Composable 内部**（而非工具栏层级），以确保 `PopupPositionProvider` 收到正确的锚点坐标：

```kotlin
Box(
    contentAlignment = Alignment.Center,
    modifier = Modifier.size(64.dp).clickable { /* 切换面板显示 */ }
) {
    // 激活背景（可选）
    // 图标
    Icon(...)
    // 子面板：锚点即本 Box
    if (showPanel) {
        XxxPanel(onDismiss = { ... })
    }
}
```

## 2. 按钮通用状态规范

### 2.1 三种交互状态

工具栏所有图标按钮统一使用以下三种状态：

| 状态 | 背景 | 图标颜色 | 触控反馈 |
|------|------|---------|---------|
| **普通（Normal）** | 透明 | `color.icon.default`（`#9E9E9E`，深色模式） | Ripple 效果（`radius.sm` 圆角矩形范围） |
| **激活（Active）** | `#90CAF9`（浅蓝）圆角矩形，`radius.sm` | `#003C8F`（深蓝，浅色背景上的深色图标） | 激活背景从 `scale: 0.85→1.0`，`150ms`（见 design-tokens.md 6.2） |
| **禁用（Disabled）** | 透明 | `color.icon.disabled`（`#616161`，深色模式） | 无任何触摸反馈（无 Ripple，无动画） |

> 激活态图标颜色使用深蓝 `#003C8F` 而非白色，原因是激活背景为浅蓝 `#90CAF9`，需要深色图标保持足够对比度。

### 2.2 激活态高亮块尺寸

| 模式 | 高亮背景尺寸 | 圆角 |
|------|-----------|------|
| 大屏 | `40dp × 40dp` | `radius.sm`（8dp） |
| PAD | `40dp × 40dp` | `radius.sm`（8dp） |

### 2.3 Ripple 效果规范

- Ripple 颜色：`rgba(144, 202, 249, 0.24)`（`color.brand.primary` 的 24% 透明版本）
- Ripple 范围：触控热区边界（64dp / 48dp 圆角矩形）
- 时长：遵循 Android Material 3 默认（约 200ms 扩散 + 100ms 淡出）

## 3. 区域 A：绘图工具区

### 3.1 工具按钮列表

| 按钮 | 图标 | 互斥组 | 说明 |
|------|------|------|------|
| 画笔（Brush） | 铅笔图标（`Icons.Default.Edit`） | 是（A 区互斥） | App 启动默认激活 |
| 橡皮擦（Eraser） | 橡皮图标（自定义 `ic_eraser`） | 是 | 首次点击仅激活；已激活时再次点击弹出清除子面板 |
| 图形工具（Shape） | **当前选中图形轮廓**（Canvas 实时渲染，`ShapeRenderer`） | 是 | 点击激活同时弹出图形子面板；图标随选中图形实时更新 |
| 便利贴（Sticky Note） | 便利贴图标 | — | v1.5 占位按钮，**禁用态**，点击触发 Toast "便利贴即将推出" |

### 3.2 画笔按钮行为

- 点击：激活画笔工具，其他工具按钮切换为普通状态
- 若画笔已激活，再次点击：无响应（不收起任何面板）
- 激活后与 B 区粗细、C 区颜色联动生效

### 3.3 橡皮擦按钮行为

```
点击橡皮擦按钮：
  ├─ 若橡皮未激活：
  │    ├─ 激活橡皮工具（A 区互斥，其他工具切为普通态）
  │    ├─ B 区自动切换为橡皮擦大小选择（见第 4 节）
  │    └─ 不弹出子面板（大小直接在 B 区操作）
  │
  └─ 若橡皮已激活：
       └─ 弹出清除子面板（向上浮层，见第 9 节）
```

### 3.4 图形工具按钮行为

```
点击图形工具按钮：
  ├─ 若图形工具未激活：
  │    ├─ 激活图形工具
  │    └─ 弹出图形子面板（向上浮层，见第 10 节）
  │
  └─ 若图形工具已激活：
       ├─ 子面板已显示 → 收起子面板（图形工具保持激活）
       └─ 子面板已收起 → 弹出子面板
```

图形工具按钮图标实时反映当前选中图形：使用 `Canvas` + `ShapeRenderer.drawShape()` 在 `24dp` 尺寸内渲染当前图形轮廓（与图形子面板中的预览图标一致），激活态用深蓝 `#003C8F` 渲染，普通态用 `#9E9E9E`。

## 4. 区域 B：笔画粗细 / 橡皮擦大小区（上下文切换）

B 区根据当前激活工具自动切换内容：
- **画笔 / 图形工具激活** → 显示笔画粗细三档
- **橡皮擦激活** → 显示橡皮擦大小三档

### 4.1 模式一：笔画粗细（画笔 / 图形工具激活时）

```
[ ━细 (Thin) ]  [ ─中 (Medium) ]  [ ▬粗 (Thick) ]

  三个按钮横排，B 区内互斥
  图形工具激活时，粗细控制图形边框线宽
```

| 档位 | 图标（横线） | 线条高度 | 线条宽度 |
|------|-----------|---------|---------|
| 细（Thin） | 细横线 | `2dp` | `22dp` |
| 中（Medium） | 中横线 | `5dp` | `22dp` |
| 粗（Thick） | 粗横线 | `10dp` | `22dp` |

图标线条颜色：
- 普通状态：`color.icon.default`（`#9E9E9E`）
- 激活状态：`#003C8F`（深蓝），背景 `#90CAF9`（浅蓝）高亮块，`40dp × 40dp`，`radius.sm`

### 4.2 模式二：橡皮擦大小（橡皮擦激活时）

```
[ ○小 (Small) ]  [ ○中 (Medium) ]  [ ○大 (Large) ]

  三个按钮横排，B 区内互斥
  图标为中空圆圈，圆圈直径代表橡皮擦相对大小
```

| 档位 | 图标圆圈直径 | 实际橡皮擦半径（归一化） |
|------|-----------|---------------------|
| 小（Small） | `8dp` | 画布宽度的 `1%` |
| 中（Medium） | `14dp` | 画布宽度的 `2%` |
| 大（Large） | `20dp` | 画布宽度的 `4%` |

图标圆圈样式：
- 中空圆圈（仅边框，无填充），边框粗细随档位：小 `1.5dp`、中 `2dp`、大 `2.5dp`
- 普通状态：边框颜色 `color.icon.default`（`#9E9E9E`）
- 激活状态：边框颜色 `#003C8F`，背景 `#90CAF9` 高亮块，`40dp × 40dp`，`radius.sm`

## 5. 区域 C：颜色区

### 5.1 颜色区布局

```
[ ●黑 ]  [ ●红 ]  [ ●蓝 ]  [ ●自定义 ]  [ 🎨调色盘 ]

  4 个快捷色色块 + 1 个调色盘入口按钮
  快捷色色块为圆形（color.quickpick.*，直径 36dp，热区 64dp × 64dp）
  调色盘按钮为图标按钮（与其他工具栏按钮规格一致）
```

### 5.2 快捷色色块规范

| 槽位 | 颜色 | Token | 深色模式描边 |
|------|------|-------|-----------|
| 槽位 1（黑） | `#212121` | `color.quickpick.black` | `1dp` 白色描边（深色画布背景下黑色不可见时增加对比） |
| 槽位 2（红） | `#E53935` | `color.quickpick.red` | 无描边 |
| 槽位 3（蓝） | `#1E88E5` | `color.quickpick.blue` | 无描边 |
| 槽位 4（自定义） | 动态色（默认 `#43A047`） | `color.quickpick.custom` | 若自定义色与深色背景接近，自动加 `1dp` 白色描边 |

**色块交互状态**：

| 状态 | 视觉 |
|------|------|
| 普通（未选中） | 圆形色块，无描边（黑色除外） |
| 激活（已选中） | 圆形色块 + `3dp` 白色外描边 + `2dp` 透明间隙（形成选中圆环） |
| 按压中 | `scale: 0.9`（略微缩小，触摸反馈），`100ms` |

**色块点击行为**：
- 立即切换书写颜色，无弹窗、无动画延迟（颜色切换 0ms 即时生效）
- 选中视觉从上一个激活色块消失，新色块出现激活环
- 若当前激活的是"自定义槽"颜色，从色板选新颜色后，槽位 4 色块颜色同步更新

### 5.3 调色盘按钮规范

- 图标：调色盘 / 彩虹圆圈
- 点击：弹出颜色选择器 Popup（详见第 11 节，遵循第 0 节统一规范）
- 无激活状态（调色盘入口不作为"当前工具"存在）
- **当前颜色指示器**：图标右上角常驻一个 `10dp` 彩色实心圆点，始终显示当前书写颜色；白色半透明描边 `1dp`（保证深色背景下颜色可见）

## 6. 区域 D：操作区（撤销 / 重做）

### 6.1 按钮列表

| 按钮 | 图标 | Token 颜色（可用） | Token 颜色（禁用） |
|------|------|-----------------|-----------------|
| 撤销（Undo） | 向左弧形箭头（↩） | `color.icon.default` | `color.icon.disabled` |
| 重做（Redo） | 向右弧形箭头（↪） | `color.icon.default` | `color.icon.disabled` |

### 6.2 可用 / 禁用状态规则

| 条件 | 撤销按钮状态 | 重做按钮状态 |
|------|-----------|-----------|
| UndoStack 不为空 | 可用 | — |
| UndoStack 为空 | **禁用** | — |
| RedoStack 不为空 | — | 可用 |
| RedoStack 为空 | — | **禁用** |

**禁用态视觉过渡**：
- 从可用 → 禁用：图标颜色从 `color.icon.default` 过渡到 `color.icon.disabled`，`100ms`
- 从禁用 → 可用：反向过渡，`100ms`
- 禁用态不显示 Ripple，触摸无任何反馈

### 6.3 按钮行为

- 撤销按钮（可用态）点击：执行 UndoStack 栈顶操作，弹出至 RedoStack
- 重做按钮（可用态）点击：执行 RedoStack 栈顶操作，弹回至 UndoStack
- 撤销/重做均为**仅工具栏按钮触发**，不支持任何手势触发（避免多人书写时多点触控冲突，见 ui-ux-outline.md 1.4 节）
- 无长按加速（大屏场景用户不需要快速多步撤销）

## 7. 区域 E：功能区（协同 / 分享 / 更多）

### 7.1 按钮列表

### 11.4 快速颜色区规范

- 4 个色块，与工具栏快捷色完全一致（规格见第 5.2 节）
- 标题：`快速颜色`，`typography.size.headline`，`typography.weight.semibold`
- 标题与内容间距：`spacing.sm`（8dp）
- 作用：底部 Sheet 中可快速选中已有快捷色，无需关闭重选

### 11.5 预设颜色网格规范

- 5 列 × 4 行宫格，共 20 色（色值参考 design-tokens.md 1.6 节）
- 色块形状：圆形（`radius.full`），直径 `40dp`（大屏）/ `32dp`（PAD）
- 触控热区：`52dp × 52dp`（大屏）/ `44dp × 44dp`（PAD）
- 色块间距：`spacing.sm`（8dp）
- 选中态：同工具栏快捷色，`3dp` 白色外描边 + `2dp` 透明间隙

### 11.6 最近使用区规范

- 显示最近 5 种使用过的颜色（含从预设色和 HSV 自定义选取的颜色）
- 色块规格同预设颜色网格
- 若最近使用颜色不足 5 个，空余槽位显示为灰色虚线圆圈（不可点击）
- 最近使用颜色存储于 `SharedPreferences`，App 重启后保留

### 11.7 HSV 自定义区规范

| 组件 | 规范 |
|------|------|
| HSV 选色区 | 正方形饱和度/亮度选色板（左）+ 竖向色相滑块（右），选色板边长 `160dp`（大屏） |
| 透明度滑块 | 水平渐变条（当前色 → 透明棋盘格），高度 `24dp` |
| Hex 输入框 | 7 字符输入（`#RRGGBB`），`typography.family.monospace`，`typography.size.label` |
| 确认按钮 | 文字按钮"确认"，`color.brand.primary` 文字色，点击后选色生效并关闭底部 Sheet |

**选色生效逻辑**：
1. 用户在 HSV 区域操作时，**实时预览**当前选中色（工具栏预览区小色块）
2. 点击"确认"按钮：
   - 所选颜色替换工具栏槽位 4（自定义色）
   - 更新为当前书写颜色
   - 写入最近使用颜色列表
   - Bottom Sheet 收起
3. 若用户未点击确认直接关闭（向下滑 / 点遮罩）：**取消选色**，颜色恢复操作前的状态
